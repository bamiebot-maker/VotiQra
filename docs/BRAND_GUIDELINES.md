# VotiQra Brand Guidelines

> Version 1.0

---

# Brand Vision

VotiQra is a modern election platform built for organizations that demand trust, transparency, security, and simplicity.

Every screen should communicate confidence.

Users should feel safe casting votes.

Administrators should feel in control.

Organizations should feel proud to use the platform.

---

# Brand Personality

VotiQra should always feel:

• Professional

• Trustworthy

• Transparent

• Modern

• Premium

• Secure

• Clean

• Intelligent

• Friendly

Never feel:

• Childish

• Playful

• Loud

• Flashy

• Over-designed

• Distracting

---

# Design Philosophy

Less is more.

Whitespace is a feature.

Clarity beats decoration.

Motion should guide users.

Consistency is more important than creativity.

---

# Design Reference

Reference UI images are located inside:

assets/design-reference/

These references exist only to inspire:

- Screen layout
- White space
- Card hierarchy
- Navigation
- Information hierarchy
- Component spacing
- Mobile UX

Never copy:

- Logos
- Colors
- Icons
- Branding
- Typography
- Images

Always create original interfaces.

---

# Visual Style

The application should feel similar to modern SaaS products.

Characteristics:

Large cards

Rounded corners

Soft shadows

Large illustrations

Minimal borders

Comfortable spacing

Premium typography

Large buttons

Readable forms

High accessibility

# Background

Very Light Gray

Cards should always stand out from the background.

---

# Cards

Cards are the primary container.

Requirements:

Large radius

Soft shadow

Generous padding

Comfortable spacing

Never overcrowd cards.

---

# Buttons

Primary

Secondary

Outline

Ghost

Destructive

Loading

Disabled

Buttons should have:

Large touch targets

Rounded corners

Consistent spacing

Strong visual hierarchy

---

# Inputs

Inputs should feel spacious.

Large radius.

Clear labels.

Helpful validation.

Large touch targets.

---

# Typography

Use Inter.

Hierarchy:

Display

Heading

Title

Subtitle

Body

Caption

Button

Never use decorative fonts.

---

# Icons

Use Lucide React Native.

Icons should be:

Simple

Consistent

Recognizable

Never mix icon libraries.

---

# Illustrations

Illustrations should be:

Flat

Minimal

Friendly

Professional

Diverse

Accessible

Avoid:

3D

Cartoons

Anime

Gaming

Neon

Cyberpunk

---

# Animations

Animations should be:

Fast

Purposeful

Subtle

Professional

Preferred animations:

Fade

Scale

Slide

Avoid excessive bounce.

---

# Shadows

Use soft elevation.

Never use heavy dark shadows.

---

# Spacing

Follow the 8-point grid.

Never use random spacing.

---

# Accessibility

Support:

Dark Mode

Dynamic Text

Screen Readers

44x44 touch targets

High Contrast

Color Blind Accessibility

Keyboard Navigation (future web)

---

# Mobile First

Design everything for mobile.

Tablet support is secondary.

Desktop support will be implemented later.

---

# Future Expansion

The design system must scale to:

Admin Dashboard

Organization Portal

Web Platform

Tablet Application

Desktop Dashboard

without redesign.

---

# Brand Promise

VotiQra helps organizations conduct elections that are:

Secure

Transparent

Professional

Trusted

Accessible

Every design decision should reinforce this promise.
